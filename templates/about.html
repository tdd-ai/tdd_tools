About TDD Text Tools

This tool deascifies given texts.
The tool uses turkish-deasciifier by Emre Sevinç.
This tools calculates the frequency for tpe tokens in given text. The frequency is calculated case sensitively which means "git" and "GİT" considered as two different tokens.
This tools calculates the frequency for tpe tokens in given text. The frequency is calculated case insensitively which means "git" and "GİT" considered as one and same token.
