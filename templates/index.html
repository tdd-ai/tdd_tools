Frequency
Case Sensitive Frequency Calculator

This tool calculates the frequency of tokens in given text. The frequency is calculated case sensitively which means "git" and "GİT" considered as two different tokens.

Frequency
Case Insensitive Frequency Calculator

This tool calculates the frequency of tokens in given text. The frequency is calculated case insensitively which means "git" and "GİT" considered as one and same token.

This tools checks if words in a given list occurs in the given text file.
